Integrasi API

Neptune API
https://zelapioffciall.koyeb.app
API yang digunakan untuk fitur TQTO (Terima Kasih To) pada bot WhatsApp, menyediakan berbagai layanan web yang dapat diakses melalui permintaan HTTP. API ini dikembangkan dan dimiliki oleh Hazel.

  • Integrasi mudah dengan Node.js
  • Dokumentasi API lengkap

Pemilik dan pengembang Neptune API yang menyediakan layanan REST API untuk berbagai kebutuhan bot WhatsApp, termasuk fitur TQTO yang digunakan dalam Nime Bot.

Kontribusi:

Proyek Unggulan

Nime - Assisten

Bot WhatsApp serbaguna yang dirancang untuk membantu berbagai tugas, mulai dari informasi, hiburan, hingga produktivitas. Bot ini telah digunakan oleh 200+ pengguna aktif di grup WhatsApp.

Bot ini dibangun menggunakan library @yupra/bailes yang merupakan fork dari Baileys dengan peningkatan fitur dan performa. Dijalankan pada Node.js 20+ untuk memastikan stabilitas dan kompatibilitas dengan fitur JavaScript terbaru.

Bot juga mengintegrasikan Neptune API yang dikembangkan oleh Hazel untuk fitur TQTO (Terima Kasih To) yang memungkinkan pengguna untuk mengirimkan ucapan terima kasih secara otomatis dengan berbagai template yang tersedia.

  • 422 Fitur Lengkap
  • Respons Cepat & Ramah
  • Berbagai Fitur Menarik
  • Mudah Digunakan di Grup
  • Dapat Dikustomisasi
  • Berbasis @yupra/bailes
  • Support Node.js 20+
  • Integrasi Neptune API untuk TQTO
